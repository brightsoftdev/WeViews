FPanel Class Reference

Inherits from FUIView : UIControl
Declared in FPanel.h
FPanel.m

Overview

A container for UIViews can automatically layout its contents using a variety of layout strategies.

Subviews are added in layers (FPanelLayer).

See: FPanelLayer, FPanelLayout.
